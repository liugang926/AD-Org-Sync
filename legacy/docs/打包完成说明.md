## 部署说明

### 方式一：快速部署（推荐）

1. 将 `dist` 目录下的所有文件复制到目标服务器
2. 编辑 `config.ini` 填写配置信息
3. 运行 `Test-LDAP.exe` 和 `Test-WeCom-API.exe` 测试连接
4. 启动 `WeCom-AD-Sync.exe`

### 方式二：最小部署

如果只需要主程序：
1. 复制 `WeCom-AD-Sync.exe`
2. 复制 `config.ini`
3. 创建 `logs` 目录
