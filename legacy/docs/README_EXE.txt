================================================================================
        企业微信-AD同步工具 v2.3.0 - 使用说明
================================================================================

📦 文件说明
--------------------------------------------------------------------------------
  企业微信AD同步工具.exe          - 主程序（GUI界面）
  config.ini                      - 配置文件（必需）
  logs/                          - 日志目录（自动创建）
  sync_state.json                - 同步状态（自动创建）
  
  可选文件：
  测试LDAP连接.exe                - LDAP连接测试工具
  测试企业微信API.exe             - 企业微信API测试工具


🚀 快速开始
--------------------------------------------------------------------------------
  1. 准备配置文件
     - 确保 config.ini 在 exe 同目录下
     - 填写企业微信和LDAP配置信息
     
  2. 首次运行建议先测试连接
     - 运行: 测试LDAP连接.exe
     - 运行: 测试企业微信API.exe
     
  3. 运行同步程序
     - 双击: 企业微信AD同步工具.exe
     - 点击"开始同步"
     

⚙️ 配置说明
--------------------------------------------------------------------------------
  config.ini 必须包含以下配置：
  
  [WeChat]
  corpid = 企业ID
  agentid = 应用ID
  corpsecret = 应用Secret
  
  [LDAP]
  server = AD服务器地址
  domain = 域名
  username = 管理员账号
  password = 管理员密码
  usessl = true
  port = 636
  
  [Sync]
  baseDN = DC=example,DC=com
  

📋 功能特性
--------------------------------------------------------------------------------
  ✅ 自动同步企业微信组织架构到AD
  ✅ 支持全量同步和增量同步
  ✅ 自动创建OU和安全组
  ✅ 自动创建/更新用户账户
  ✅ 自动禁用已离职用户
  ✅ 详细的操作日志和报告
  ✅ 友好的GUI界面
  ✅ 支持定时同步
  ✅ 支持随时取消
  

📊 日志文件
--------------------------------------------------------------------------------
  程序会自动在 logs/ 目录创建以下日志：
  
  - ad_wecom_sync_YYYYMMDD_HHMMSS.log          - 常规日志
  - ad_wecom_sync_detailed_YYYYMMDD_HHMMSS.log - 详细日志
  - sync_operations_YYYYMMDD_HHMMSS.csv        - 操作记录（CSV）
  - sync_validation_YYYYMMDD_HHMMSS.txt        - 数据验证报告
  - disabled_accounts_YYYYMMDD_HHMMSS.csv      - 禁用账户清单
  

🐛 常见问题
--------------------------------------------------------------------------------
  Q1: 双击exe没反应？
  A1: 可能被杀毒软件拦截，请添加到白名单
  
  Q2: 提示找不到config.ini？
  A2: 确保config.ini和exe在同一目录
  
  Q3: 连接失败？
  A3: 先运行测试工具诊断问题
      - 测试LDAP连接.exe
      - 测试企业微信API.exe
  
  Q4: 如何查看详细错误？
  A4: 使用控制台版本，可以看到详细日志
      或者查看 logs/ 目录中的日志文件
  
  Q5: 同步很慢？
  A5: 首次同步较慢，后续可使用增量同步模式
  

🔒 安全注意
--------------------------------------------------------------------------------
  ⚠️ config.ini 包含敏感信息（密码）
  
  建议：
  1. 设置文件权限，只允许管理员访问
  2. 不要将config.ini提交到代码仓库
  3. 确保使用LDAPS（SSL加密）
  4. 在企业微信后台配置IP白名单
